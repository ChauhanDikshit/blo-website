How to contribute

Contributions are welcome, especially adding missing papers, improving tags, and fixing references.

  1. Add a paper: edit data/papers.json and add a new entry (title/authors/venue/year/tags/tier/url).
  2. Improve taxonomy: edit data/taxonomy.json to add/expand nodes (keep IDs stable).
  3. Fix a page: open a Pull Request to edit the HTML pages in the root folder.
  4. Quick feedback: open a GitHub Issue describing the change you want.

Suggested rule: keep method_family broad (Classical/Evolutionary/ML-based BLO/Hybrid/Survey) and use tags for fine categories (e.g., kkt, surrogate, implicit-diff, nas).
