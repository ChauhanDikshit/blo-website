How to add papers

  • Add entries to data/papers.json (one object per paper).
  • Use stable tags aligned with the Taxonomy page (e.g., ml.hpo, ea.nested).
  • Keep method family and problem type consistent (filters are built from observed values).

Minimal JSON template:

[
  {
    "year": 2024,
    "title": "Paper title here",
    "authors": "Author A; Author B; Author C",
    "venue": "Journal/Conference",
    "tier": "A",
    "family": "ml",
    "ptype": "single-follower",
    "tags": ["ml.hpo", "differentiable", "implicit-diff"],
    "links": {
      "doi": "",
      "pdf": "",
      "code": ""
    }
  }
]

Tip: keep family coarse (e.g., classical, ea, ml, hybrid) and use tags for finer details.
