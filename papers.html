How to add papers

  • Add one JSON object per paper to data/papers.json.
  • authors must be an array (because the site prints authors.join(", ")).
  • Keep method_family and problem_type consistent; dropdowns are built from existing values.
  • Use tags for finer details (e.g., ml.hpo, ea.nested, kkt).

Minimal entry template (matches assets/app.js):

{
  "year": 2024,
  "title": "Paper title here",
  "authors": ["Author A", "Author B", "Author C"],
  "venue": "Journal/Conference",
  "tier": "A",
  "method_family": "ML-based BLO",
  "problem_type": "Continuous BLO",
  "tags": ["implicit differentiation", "hyperparameter optimization"],
  "url": "https://doi.org/...",
  "note": "Optional short note."
}

Tip: use broad method families like Classical, Evolutionary, ML-based BLO, Hybrid, Survey.
